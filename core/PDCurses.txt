PDCurses Portable Core and for Windows console
======================
Welcome to PDCurses!
====================

The files in this directory are released to the public domain.
PDCurses is an implementation of X/Open curses for multiple platforms. Many of the files included in PDCurses are in the public domain. 
The latest version and more details on the licensing information can be found at:

   https://pdcurses.org/
